# NFT Trader (Portals)

Мультиаккаунтный трейдер для Portals с оферами, ордерами, автопродажей и Telegram-мониторингом.

## Запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python .\main.py
```

Лончер: `configs/snipers.json`

Прямой запуск Portals-сервиса:

```bash
python .\portal_sniper.py
```

или:

```bash
python -m src.services.portal.sniper
```

## Конфиги

- Стратегии и правила: `src/services/portal/config/strategy.json`
- Мультиаккаунты: `configs/portal_accounts.json`
- Лончер рынков: `configs/snipers.json`

## Переменные окружения (.env)

- Лончер автоматически читает `.env` в корне проекта.
- Для аккаунтов через `auth_env` используйте `PORTAL_AUTH`, `PORTAL_AUTH_ALT_1`, и т.д.
- Для Telegram используйте `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_IDS` (через запятую), `TELEGRAM_ENABLED=true`.

## Что реализовано

- Поиск выгодных ордеров/офферов в диапазонах цен.
- Автоперебивание ордеров и офферов шагом `0.01`, если сделка остается выгодной.
- Бесконечный режим и параллельная работа нескольких аккаунтов.
- Личные селекторы по подарку, модели и фону (в любых комбинациях).
- Отдельные правила для «только что выставленных» (recent).
- Защита от неликвида через фильтры ликвидности.
- Опциональная автоотмена ордеров/офферов/листингов по TTL.
- Автопродажа с учетом модели/фона.
- Репрайс листинга на `0.01` ниже флора с учетом ограничений.
- Telegram-бот на `aiogram` со статистикой и прибылью.
